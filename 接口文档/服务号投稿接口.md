> post

####参数：

参数 | 必选 | 类型 | 说明
- | :-: | :-: | -
serviceNumberContent | true | com.zsx.model.pojo.ServiceNumberContent | 服务号投稿实体类

####权限值：
> 

####返回示例

	{
	    "code": 1,
	    "message": "成功"
	}
