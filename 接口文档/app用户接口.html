新增用户

请求url:

/api/user/add

请求方式:

post

参数：

参数 必选 类型 说明
User true com.zsx.model.pojo.User 用户实体类

权限值：

app:add

返回示例

{
    "code": 1,
    "message": "成功"
}

统计新增用户数量

请求url:

/api/user/statistic/new/user/{dayNumber}/{type}/{isAll}

请求方式:

post

参数：

参数 必选 类型 说明
dayNumber true Integer 1：昨日 7：七天 30：一月
type true Integer 0：all 1：ios 2：android
isAll true Integer 0：否 1：是

权限值：

userCount

返回示例

{
    "code": 1,
    "data": 0,
    "message": "SUCCESS"
}
