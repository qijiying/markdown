##新增简报加推送


####请求url: 
> /api/web/briefing/add/{type}

####请求方式: 
> post

####参数：

参数 | 必选 | 类型 | 说明
- | :-: | :-: | -
Briefing | true | com.zsx.model.pojo.Briefing | 简报实体类
type | true | Integer | 类型  1：保存 2：立即发布

####权限值：
> easypaper:add

####返回示例

	{
	    "code": 1,
	    "message": "成功"
	}

<span id="data_grid"></span>
##简报列表


####请求url: 
> /api/web/briefing/dataGrid

####请求方式: 
> get

####参数：

参数 | 必选 | 类型 | 说明
- | :-: | :-: | -
current | false | Integer | 当前页码
size | false | Integer | 每页条数 

####权限值：
> easypaper:dataGrid

####返回示例

	{
    "code": 1,
    "message": "成功",
    "pages": 1,
    "total": 1,
    "current": 1,
    "data": [
        {
          。。。  
        }
    ],
    "size": 10
	}

##修改简报


####请求url: 
> /api/web/briefing/update/{type}

####请求方式: 
> post

####参数：

参数 | 必选 | 类型 | 说明
- | :-: | :-: | -
Briefing | true | com.zsx.model.pojo.Briefing | 简报实体类
type | true | Integer | 1：仅保存  2：立即发布

####权限值：
> easypaper:edit

####返回示例

	{
	    "code": 1,
	    "message": "成功"
	}



<span id="delete"></span>
##删除简报


####请求url: 
> /api/web/briefing/delete

####请求方式: 
> post

####参数：

参数 | 必选 | 类型 | 说明
- | :-: | :-: | -
ids | true | Integer[] | 简报id数组

####权限值：
> easypaper:delete

####返回示例

	{
	    "code": 1,
	    "message": "成功"
	}
