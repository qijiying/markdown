新增权限

请求url:

/api/web/sys/perm/add

请求方式:

post

参数：

参数 必选 类型 说明
SysPerm true com.zsx.model.pojo.SysPerm 权限实体类

权限值：

authority_index:add

返回示例

{
    "code": 1,
    "message": "成功"
}
