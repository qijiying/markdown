##新增软文

####请求url: 
> /api/web/news/add

####请求方式: 
> post

####参数：

参数 | 必选 | 类型 | 说明
- | :-: | :-: | -
news | true | com.zsx.model.pojo.News | 软文实体类

####权限值：
> softtest:add

####返回示例

	{
	    "code": 1,
	    "message": "成功"
	}

##服务号类别列表

####请求url: 
> /api/web/news/dataGrid

####请求方式: 
> post

####参数：

参数 | 必选 | 类型 | 说明
- | :-: | :-: | -
current | false | Integer | 当前页码
size | false | Integer | 每页条数 

####权限值：
> softtext:dataGrid

####返回示例

	{
	    "code": 1,
	    "message": "成功",
	    "pages": 1,
	    "total": 1,
	    "current": 1,
	    "data": [
	        {

	        }
	    ],
	    "size": 10
	}
