##新增代理商


####请求url: 
> /api/web/agent/add

####请求方式: 
> post

####参数：

参数 | 必选 | 类型 | 说明
- | :-: | :-: | -
AgentInfo | true | com.zsx.model.pojo | 代理商实体类
areaCode | true | Integer[] | 区域code
areaType | true | Integer | 区域类型（1:省 2:市 3:区） 

####权限值：
> agent:add

####返回示例

	{
	    "code": 1,
	    "message": "成功"
	}
