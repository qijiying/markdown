<span id="add"></span>
##新增角色

####请求url: 
> /api/web/sys/role/add

####请求方式: 
> post

####参数：

参数 | 必选 | 类型 | 说明
- | :-: | :-: | -
sysRole | true | com.zsx.model.pojo.SysRole | 角色实体类

####权限值：
> role:add

####返回示例

	{
	    "code": 1,
	    "message": "成功"
	}
