##新增版本


####请求url: 
> /api/web/appversion/add

####请求方式: 
> post

####参数：

参数 | 必选 | 类型 | 说明
- | :-: | :-: | -
AppVersion | true | com.zsx.model.pojo.AppVersion | 版本实体类

####权限值：
> app:add

####返回示例

	{
	    "code": 1,
	    "message": "成功"
	}
