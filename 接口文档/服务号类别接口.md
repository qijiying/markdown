##新增服务号

####请求url: 
> /api/web/servicenumber/class/add

####请求方式: 
> post

####参数：

参数 | 必选 | 类型 | 说明
- | :-: | :-: | -
serviceNumberClass | true | com.zsx.model.pojo.ServiceNumberClass | 服务号类别实体类

####权限值：
> 

####返回示例

	{
	    "code": 1,
	    "message": "成功"
	}
